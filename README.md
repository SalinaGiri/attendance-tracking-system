# Attendance Tracking System

A full-stack web app built for **UC Leuven-Limburg as a real client**, letting lecturers track and navigate student attendance across large classes — with sorting, search, and visual reporting.

Built by a team of student developers over a semester, following **Scrum / Agile** with sprints, daily stand-ups, and client reviews.

## What it does

- Lets lecturers browse and manage long lists of students quickly
- Sorting and search to filter attendance records
- Visual reporting with graphs to spot attendance trends at a glance
- Responsive interface designed around how lecturers actually work
